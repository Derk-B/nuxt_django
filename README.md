# Web
Install packages first:
```bash
web/> npm install
```

Run webapplication using: 
```bash
web/> npm run dev
```

# Api
Install packages first:
```bash
api/> pip install pipenv
api/web_api/> pipenv run pip install django
```

Run using: 
```bash
api/web_api> pipenv run python3 manage.py runserver
```
